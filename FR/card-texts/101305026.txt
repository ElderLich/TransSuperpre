[name]
Gray Layer

[desc]
Vous pouvez Sacrifier cette carte et révéler 1 monstre Machine dans votre main ; ajoutez 1 monstre Machine depuis votre Deck à votre main d'Attribut différent du monstre révélé, puis si vous avez ajouté un monstre avec max. 1200 ATK, vous pouvez Invoquer Spécialement 1 monstre Machine depuis votre main.
Si un ou plusieurs monstres en votre possession sont détruits par un effet de carte, tant que cette carte est dans votre Cimetière : vous pouvez bannir cette carte ; détruisez 1 carte sur le Terrain.
Vous ne pouvez utiliser chaque effet de "Gray Layer" qu'une fois par tour.
