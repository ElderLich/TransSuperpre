[name]
Magistrat Malveillant Désenchaîné de Kamura

[desc]
Vous pouvez cibler max. 3 cartes que vous contrôlez ; détruisez-les, et si vous le faites, Invoquez Spécialement cette carte depuis votre main, puis vous pouvez appliquer n'importe lequel des effets suivants, dans l'ordre, selon les types d'origine des cartes détruites.
● Monstre : Piochez 1 carte.
● Magie : Changez la position de combat d'1 monstre sur le Terrain.
● Piège : Détruisez 1 autre carte sur le Terrain.
Si cette carte sur le Terrain est détruite par un effet de carte : vous pouvez détruire tous les monstres sur le Terrain. Vous ne pouvez utiliser chaque effet de "Magistrat Malveillant Désenchaîné de Kamura" qu'une fois par tour.
